¿Cómo publicar masivamente?

Paso 1: Descarga la planilla CSV (Debes contar con Microsoft Excel, sino posees Descarga aquí)

Paso 2: Ingresa los datos de tus anuncios en la planilla CSV

Paso 3: Sube el CSV con los datos de tus anuncios

El archivo tendrá un proceso de verificación, nos comunicaremos a tu correo al ser aprobado

Instrucciones para cargar datos en el CSV:

-Dejar vacías las columnas con nombre space

-Las columnas con nombre first_registration_month_year e inspection_valid_until_month_year deben cumplir con el formato mm.yyyy ejemplo: 06.2016

-La Columna price_contains_vat solo recibe valor numérico de 0 y 1 0 Para indicar que el precio no posee iva 1 Para indicar que el precio contiene iva

- Las siguientes columnas deben contener números enteros (ej: 1,1000,1500) son: price, mileage, doors, seats, engine_displacement, power_hp
